# 1. GCP 계정 생성 및 무료 크레딧 활성화

이 문서는 Google Cloud Platform(GCP) 계정을 생성하고 $300 무료 크레딧을 활성화하는 방법을 안내합니다.

---

## 용어 설명

| 용어 | 설명 |
|------|------|
| **GCP** | Google Cloud Platform. 구글의 클라우드 서비스 |
| **프로젝트** | GCP에서 리소스를 묶는 단위. 하나의 프로젝트 안에 VM, 스토리지 등이 포함됨 |
| **프로젝트 이름** | 사람이 읽기 쉬운 표시용 이름 (예: "내 OpenClaw 프로젝트") |
| **프로젝트 ID** | 시스템에서 사용하는 고유 식별자 (예: `my-openclaw-project-12345`). **변경 불가** |

---

## 사전 준비

- Google 계정 (Gmail)
- 신용카드 또는 체크카드 (인증용)

> **카드 등록이 걱정되시나요?**
>
> 걱정하지 마세요:
> - 무료 체험 기간(90일) 동안 **절대 청구되지 않습니다**
> - 크레딧이 소진되어도 **자동으로 유료 전환되지 않습니다**
> - 직접 "업그레이드" 버튼을 누르지 않는 한 과금 없음
> - 카드 등록은 봇/스팸 방지용 본인 인증 목적

---

## Step 1: GCP Console 접속

1. 브라우저에서 [cloud.google.com](https://cloud.google.com) 접속

2. 화면 우측 상단을 보세요:
   ```
   ┌─────────────────────────────────────────────────────┐
   │  Google Cloud          [제품▼] [솔루션▼] [가격]    │
   │                                    [Console] [무료로 시작하기] ← 여기!
   └─────────────────────────────────────────────────────┘
   ```

3. **"무료로 시작하기"** (파란 버튼) 클릭
   - 또는 이미 계정이 있다면 **"Console"** 클릭

4. Google 계정으로 로그인

---

## Step 2: 무료 체험 시작

1. **"무료로 시작하기"** 버튼 클릭

2. 국가/지역 선택:
   ```
   국가: [대한민국 ▼]
   ```

3. 서비스 약관 동의 체크박스 2개 체크

4. **"계속"** 클릭

---

## Step 3: 결제 정보 입력

> **다시 한번 강조**: 이 단계에서 카드를 등록해도 **무료 체험 중 청구되지 않습니다**.
> $1 정도의 소액이 임시로 결제되었다가 바로 취소됩니다 (카드 유효성 확인용).

1. **계정 유형**: `개인` 선택 (사업자가 아니라면)

2. **이름 및 주소** 입력:
   - 카드에 등록된 이름과 동일하게 입력
   - 주소는 카드 청구지 주소

3. **결제 수단 추가**:
   - 신용카드 또는 체크카드 정보 입력
   - 해외 결제가 가능한 카드 필요 (대부분의 카드 가능)

4. **"무료 평가판 시작"** 클릭

---

## Step 4: 무료 크레딧 확인

1. [GCP Console](https://console.cloud.google.com) 접속

2. 좌측 상단 햄버거 메뉴(☰) 클릭 → **"결제"** 클릭
   ```
   ┌──────────────────┐
   │ ☰ Google Cloud   │
   ├──────────────────┤
   │ > 대시보드        │
   │ > Compute Engine │
   │ > Cloud Storage  │
   │ > 결제          ← 여기 클릭
   │ > IAM 및 관리자  │
   └──────────────────┘
   ```

3. **"크레딧"** 탭에서 잔액 확인

| 항목 | 값 |
|------|-----|
| 무료 크레딧 | **$300** (약 40만원) |
| 유효 기간 | 90일 |
| 자동 청구 | **없음** (업그레이드 전까지) |

---

## Step 5: 첫 번째 프로젝트 생성

1. 화면 상단의 프로젝트 선택 영역 클릭:
   ```
   ┌─────────────────────────────────────────┐
   │ ☰  [My First Project ▼]  검색...       │
   │         ↑ 여기 클릭                     │
   └─────────────────────────────────────────┘
   ```

2. 팝업에서 **"새 프로젝트"** 클릭

3. 프로젝트 정보 입력:
   ```
   프로젝트 이름: [my-openclaw-project    ]  ← 원하는 이름
   프로젝트 ID:   my-openclaw-project-12345  ← 자동 생성됨
                  [수정] 버튼으로 변경 가능
   위치:         [조직 없음]
   ```

4. **"만들기"** 클릭

> **프로젝트 ID 중요!**
> - 프로젝트 ID는 전 세계에서 고유해야 함
> - **한번 생성하면 변경 불가**
> - 나중에 명령어에서 자주 사용됨
> - 예: `gcloud config set project my-openclaw-project-12345`

---

## 프로젝트 ID 확인하는 법

나중에 프로젝트 ID가 필요할 때:

**방법 1**: Console에서 확인
```
상단 프로젝트 선택 클릭 → 프로젝트 목록에서 ID 컬럼 확인
```

**방법 2**: gcloud 명령어 (CLI 설치 후)
```bash
gcloud projects list
```

**방법 3**: 프로젝트 설정 페이지
```
좌측 메뉴 → IAM 및 관리자 → 설정 → 프로젝트 ID
```

## 무료 크레딧 사용 팁

### 비용 절약 방법

| 방법 | 설명 | 절약률 |
|------|------|--------|
| **VM 중지** | 사용 안 할 때 VM 중지 | 컴퓨팅 비용 100% 절약 |
| **Spot VM** | 중단 가능성 있지만 저렴 | 60-90% 저렴 |
| **서울 리전** | asia-northeast3, 접속 빠름 | - |
| **예산 알림** | 설정한 금액 도달 시 알림 | - |

### 무료 크레딧으로 가능한 것

$300으로 약 2-3개월간 다음 환경 운영 가능:

| 리소스 | 사양 | 월 예상 비용 |
|--------|------|-------------|
| VM | e2-standard-4 (4 vCPU, 16GB) | ~$100 |
| 디스크 | SSD 50GB | ~$8 |
| 네트워크 | 기본 사용량 | ~$5 |
| **합계** | | **~$113/월** |

## 문제 해결

### "결제 계정을 만들 수 없습니다"

**원인**: 이미 무료 체험을 사용한 계정

**해결**:
- 새 Google 계정 생성 후 시도
- 또는 기존 계정에서 유료로 업그레이드

### 카드 인증 실패

**원인**: 해외 결제 차단 또는 한도 부족

**해결**:
- 카드사 앱에서 해외 결제 허용 확인
- 카드 한도 확인 (최소 $1 필요)
- 다른 카드로 시도

### 프로젝트가 안 보임

**원인**: 다른 계정으로 로그인됨

**해결**:
1. 우측 상단 프로필 아이콘 클릭
2. 올바른 계정인지 확인
3. 상단 프로젝트 선택 → "모든 프로젝트" 탭

### "API를 사용할 수 없습니다"

**원인**: 결제 계정이 프로젝트에 연결되지 않음

**해결**:
1. 좌측 메뉴 → 결제
2. "프로젝트 연결" 클릭
3. 결제 계정 선택
