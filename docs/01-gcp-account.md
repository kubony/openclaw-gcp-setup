# 1. GCP 계정 생성 및 무료 크레딧 활성화

이 문서는 Google Cloud Platform(GCP) 계정을 생성하고 $300 무료 크레딧을 활성화하는 방법을 안내합니다.

---

## 사전 준비

- Google 계정 (Gmail)
- 신용카드 또는 체크카드 (인증용, 무료 체험 중 청구되지 않음)

---

## Step 1: GCP Console 접속

1. 브라우저에서 [cloud.google.com](https://cloud.google.com) 접속
2. 우측 상단 **"Console"** 또는 **"무료로 시작하기"** 클릭
3. Google 계정으로 로그인

---

## Step 2: 무료 체험 시작

1. **"무료로 시작하기"** 버튼 클릭
2. 국가 선택: **대한민국**
3. 서비스 약관 동의 체크
4. **"계속"** 클릭

---

## Step 3: 결제 정보 입력

> **참고**: 무료 체험 기간 동안 실제 청구되지 않습니다. 크레딧 소진 후에도 자동 결제되지 않으며, 업그레이드 전까지 유료 전환되지 않습니다.

1. 계정 유형: **개인** 선택
2. 이름, 주소 입력
3. 결제 수단 추가:
   - 신용카드 또는 체크카드 정보 입력
   - 소액 임시 결제 후 취소됨 (인증 목적)
4. **"무료 평가판 시작"** 클릭

---

## Step 4: 무료 크레딧 확인

1. [GCP Console](https://console.cloud.google.com) 접속
2. 좌측 메뉴 → **"결제"** 클릭
3. **"크레딧"** 탭에서 잔액 확인

| 항목 | 값 |
|------|-----|
| 무료 크레딧 | $300 (약 40만원) |
| 유효 기간 | 90일 |
| 자동 청구 | 없음 (업그레이드 전까지) |

---

## Step 5: 첫 번째 프로젝트 생성

1. 상단 프로젝트 선택 드롭다운 클릭
2. **"새 프로젝트"** 클릭
3. 프로젝트 이름 입력 (예: `my-openclaw-project`)
4. **"만들기"** 클릭

> **팁**: 프로젝트 ID는 나중에 변경할 수 없으니 신중하게 선택하세요.

---

## 무료 크레딧 사용 팁

### 비용 절약 방법

| 방법 | 설명 |
|------|------|
| **Spot VM 사용** | 일반 VM 대비 60-90% 저렴 |
| **필요시만 실행** | VM 중지 시 컴퓨팅 비용 없음 |
| **서울 리전** | asia-northeast3, 국내 접속 빠름 |
| **예산 알림 설정** | 결제 → 예산 및 알림에서 설정 |

### 무료 크레딧으로 가능한 것

$300으로 약 2-3개월간 다음 환경 운영 가능:

| 리소스 | 사양 | 월 예상 비용 |
|--------|------|-------------|
| VM | e2-standard-4 (4 vCPU, 16GB) | ~$100 |
| 디스크 | SSD 50GB | ~$8 |
| 네트워크 | 기본 사용량 | ~$5 |
| **합계** | | **~$113/월** |

## 문제 해결

### "결제 계정을 만들 수 없습니다"

- 이미 무료 체험을 사용한 계정일 수 있음
- 새 Google 계정으로 시도

### 카드 인증 실패

- 해외 결제가 가능한 카드인지 확인
- 카드 한도 확인 (소액 임시 결제)

### 프로젝트가 안 보임

- 상단 프로젝트 선택 → "모든 프로젝트" 탭 확인
- 올바른 계정으로 로그인했는지 확인
